# npm 与 cnpm 要点

npm i

在国内环境下，有时候 npm 下载慢或者下载失败，我们建议是使用 taobao 源，而不是使用 cnpm。

npm install --registry=https://registry.npm.taobao.org

TIP

cnpm install 或者 update 它的包都是一个 link，会有各种诡异的 bug，而且用 cnpm 安装的包 在 node_modules 里面的目录不是以包的形式存在的。
